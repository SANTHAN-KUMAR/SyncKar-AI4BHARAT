To the Judges

This AWS deployment is built using `docker-compose`. The Mock Systems container simulates legacy department databases. SyncKar runs in a completely separate container and has zero direct access to the mock databases. They interact strictly via HTTP APIs and Kafka, exactly mirroring real-world conditions.
